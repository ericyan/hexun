# hexun

A simple wrapper for [Hexun](http://www.hexun.com/)'s undocumented fund data API.

## Installation

Add this line to your application's Gemfile:

    gem 'hexun'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hexun
